## 一、她是谁

**Sakura Yuki**（纱久良雪希）是一个运行在 **QQ 平台**上的智能机器人。名字取自日文中「桜（樱）」与「雪」的意象——樱花代表温暖与美好，初雪象征纯净与宁静。

我们希望 Sakura Yuki 能像春日飘落的樱花花瓣一样，自然地融入你的日常，为群聊带来一份轻柔的陪伴。

- **生日** — 4月27日
- **代表色** — 樱花粉
- **喜欢** — 甜品、你
- **平台** — QQ
- **后端引擎** — 生巧慕斯（Namachokoretomusu）

---

## 二、核心功能

### 🌸 陪伴交流

自然流畅的对话体验，理解上下文语境，提供有温度的回应。Sakura Yuki 不只是冷冰冰的问答机器，她会用心倾听，用温柔回应。

### 💬 群聊互动

欢迎新成员、维护群规、管理成员角色——让社群运营更轻松。她会自然地融入群聊氛围，成为大家庭的一员。

### ☀️ 日常帮助

查询天气、搜索信息，日常所需一应俱全。需要什么，尽管开口。

## 三、后端引擎——生巧慕斯

> *生チョコレートムース — Namachokoretomusu*
>
> 如生巧克力般细腻丝滑，如抹茶慕斯般清新优雅。

Sakura Yuki 的后端引擎由 **Shiro Mizu** 独立开发，以「生巧克力慕斯」为名，寓意如甜品般细腻丝滑的运行体验。

### 为什么叫生巧慕斯？

名字背后的寓意，正是对架构最初的期许：

- **生巧克力慕斯** — 柔软而稳定，细腻却有层次
- 正如架构一样——外表温润柔和，内里结构分明，每一层都恰到好处地承载着自己的职责

### 技术细节

- **编程语言** — C#
- **协议支持** — OneBot V11（Satori 版本开发中）
- **数据库** — PostgreSQL
- **设计理念** — 细腻、稳定、可扩展
- **开发状态** — 底层稳定，持续迭代中
- **AI 辅助** — 注释、逻辑审查

### 核心特性

- **高可维护性** — 清晰的模块化架构，遵循 SOLID 原则，代码结构一目了然
- **高扩展性** — 插件化设计，松耦合架构，新功能可以无缝接入
- **现代化设计** — 采用最新的 C# 语言特性与 .NET 生态，异步编程、类型安全

### 系统架构

**展现层** → QQ 聊天页面
⬇
**通讯层** → WebSocket / HTTP/HTTPS
⬇
**服务层** → Sakura Yuki Core（大模型操作、消息处理、链路跟踪、配置管理）
⬇
**功能模块** → 各业务功能
⬇
**数据层** → PostgreSQL

### 性能指标

- **轻量运行** — 资源占用极低，即使长时间运行也毫无压力
- **快速响应** — 消息处理迅速，让每一次交互都自然流畅
- **稳定服务** — 即使在极端情况下也能保持稳定，值得信赖

## 六、开始使用

1. **添加机器人** — 在 QQ 中搜索 Sakura Yuki 并添加为好友，或通过邀请链接直接拉入群聊
2. **授予权限** — 在群设置中将 Sakura Yuki 设为管理员，以启用完整的群组管理功能
3. **开始互动** — 在群聊中 @Sakura Yuki 或发送指令前缀，即可开始使用各项功能
